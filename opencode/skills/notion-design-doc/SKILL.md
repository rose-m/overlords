Use the notioner sub-agent in order to create a fresh Design Doc.

* All design docs live in this database: https://www.notion.so/15a777a7099380c7aa8df298f02a4e4e
* When you create a new page in the database it will be filled with a template structure
* Stick to the rough outline of the Design Doc Template
* Set the current user as author of the document
* Be clear and precise, do not include long-stretched testing or phased migration plans unless explicitly asked.
* Use Mermaid diagrams to visualize relationships
